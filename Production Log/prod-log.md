# June 2026

I first believed that I could setup a system running proxmox alongside windows; that is, Windows would not be erased. That is not how most operating systems operate. After wiping windows off of a computer, I tried to use *DMDE* which was incredibly fascinating. I had dug through disk management on Windows and learned some diskpart in order to understand the contents of my single HDD drive. 

That is when I decided to purchase a new bare metal computer strictly for running Proxmox. I got a Dell Optiplex 9010 (16GB RAM, 3rd gen i7) for the job. The main specification I was looking for was >= 8GB RAM and >= 500GB storage.
